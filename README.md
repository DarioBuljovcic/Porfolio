# Next.js Portfolio Migration

This folder contains the Next.js migration of the original PHP portfolio from `../static_site`.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Routes

- `/`
- `/student`
- `/o-meni`
- `/projekti`
- `/vestine`
- `/blog`
- `/linkovi`
- `/kontakt`
